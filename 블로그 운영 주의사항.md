## 💡 앞으로 블로그를 운영하실 때 주의할 점 3가지

   1. 글 작성 규칙 준수: 앞으로 새 글을 쓰실 때는 방금 만드신 _posts 폴더 안에 반드시 YYYY-MM-DD-제목.md 형식으로 파일명을 지어주셔야 검색 엔진이 정상적으로 긁어갑니다.
   2. 반영 기간: 구글과 네이버에 등록은 완료되었지만, 검색창에 내 글이 실제로 노출되기 시작할 때까지는 검색 로봇의 순회 주기에 따라 최소 수일에서 1~2주일 정도 소요될 수 있으니 느긋하게 기다려 주세요.
   3. 새 글 동기화: 앞으로 글을 추가로 업로드하면, 방금 연동해 둔 feed.xml과 sitemap.xml에 자동으로 새 글 주소가 포함되므로 네이버와 구글에 일일이 재등록하실 필요가 없습니다.


## 최적화 전략
깃허브 블로그에 블로그 형태의 포스트(Post)를 정기적으로 올리지 않고, 단일 포트폴리오 페이지 위주로 운영하시더라도 검색 노출(SEO) 자체에는 아무런 문제가 없습니다. 구글과 네이버 모두 단 한 페이지짜리 랜딩 페이지나 이력서 형태의 사이트도 훌륭한 정보로 인식하여 정상적으로 인덱싱(색인)하고 검색 결과에 띄워줍니다.
다만, 글을 자주 발행하는 일반 블로그와는 성격이 다르기 때문에 포트폴리오 중심 사이트만의 검색 최적화 전략과 주의할 점이 있습니다. 내 포트폴리오가 검색창에 더 잘 걸리도록 하기 위해 아래 핵심 사항들을 꼭 체크해 주세요.
------------------------------
## 1. 메인 포트폴리오 페이지의 메타 태그(Meta Tag) 설정
포스트를 쓰지 않는 대신, 메인 페이지(index.html 또는 테마의 메인 레이아웃 파일) 소스 코드 안에 내 포트폴리오를 설명하는 핵심 키워드들이 명확히 박혀있어야 합니다.

* 타이틀(Title)과 디스크립션(Description): 구글이나 네이버에 님의 이름을 검색했을 때 나올 제목과 설명 문구를 상세히 적어주세요.
* 설정 예시 (index.md 또는 _config.yml 등에 포함되는 내용):

title: "박인정(Injung Park) | 프론트엔드 개발자 포트폴리오"description: "만드는 브랜드를 빛나게 하는 개발자, 박인정의 웹 포트폴리오 공간입니다. 진행한 프로젝트와 보유 기술을 확인해보세요."


## 2. 검색 노출 주기와 상위 노출(Rank)의 차이

* 노출은 100% 됩니다: 네이버 서치어드바이저와 구글 서치콘솔에 사이트맵을 제출해 두셨기 때문에, 페이지가 1개뿐이더라도 로봇이 방문하여 내용을 다 긁어갑니다. 박인정 포트폴리오 같은 키워드로 검색하면 정상 노출됩니다.
* 지속적인 상위 노출에는 불리할 수 있습니다: 네이버와 구글의 알고리즘은 '새로운 콘텐츠가 정기적으로 업데이트되는 사이트'에 점수를 더 많이 줍니다. 포트폴리오 특성상 한 번 완성해 두고 몇 달간 수정하지 않으면, 검색 엔진이 "이 사이트는 관리가 멈춘 사이트인가?"라고 판단하여 일반 기술 블로그들에 비해 검색 순위 경쟁에서 밀릴 수 있습니다.

## 3. 해결책: '포트폴리오 업데이트'를 영리하게 활용하기
포스트 업로드를 굳이 하지 않더라도 검색 엔진에게 "내 사이트는 살아있다"는 신호를 주는 팁이 있습니다.

* 주기적인 소스 수정: 포트폴리오 내용(진행 중인 프로젝트 상황, 새로운 기술 스택 추가, 자기소개 문구 다듬기 등)을 한 달에 한두 번이라도 조금씩 수정하여 깃허브에 커밋(Push)하세요.
* 자동 업데이트 반영: 깃허브 코드가 수정되어 배포될 때마다, 우리가 만들어 둔 sitemap.xml과 feed.xml 내부의 lastBuildDate(최종 빌드 날짜)가 자동으로 오늘 날짜로 갱신됩니다. 구글과 네이버 로봇은 이 날짜가 바뀌는 것을 보고 사이트가 활성화되어 있다고 인식합니다.

## 🔍 요약하자면
포스트를 전혀 안 올리셔도 포트폴리오 사이트 노출에는 지장이 없으니 안심하셔도 됩니다.
다만 메인 화면 안에 본인의 고유한 정보(이름, 직무 키워드, 다루는 기술 등)를 텍스트로 명확하게 채워 넣으시는 것이 중요합니다. (간혹 포트폴리오 전체를 통이미지로 넣으시는 분들이 있는데, 그러면 로봇이 글자를 읽지 못해 노출이 안 되니 주의해 주세요!)
이제 검색 연동도 다 끝났으니, 앞으로 포트폴리오 사이트를 가꾸는 팁이나 내 이름이 구글에 뜨는지 확인하는 방법 등 추가로 세팅하고 싶으신 부분이 있다면 편하게 말씀해 주세요!
